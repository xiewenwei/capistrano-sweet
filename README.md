# Capistrano::Sweet

Some useful capistrano recipes which used in boohee.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'capistrano-sweet', group: 'development'
```

And then execute:

    $ bundle

## Usage

Add `require 'capistrano/sweet'` to Capfile
